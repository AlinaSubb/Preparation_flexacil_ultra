#  Flexacil Ultra
Лендинг с формой для отправки заявок. К заказу предлагается БАД - препарат для суставов Flexacil Ultra. Обязательные поля формы - имя и телефон. Скрытые поля - subid и offerId. Под кнопками заказа формы скрипт, который автоматически меняет дату (+3 дня от текущей). В конце блока "1 curso de "Flexacil Ultra" puede contribuir a:" расположена кнопка, при нажатии на которую появляется popup.
